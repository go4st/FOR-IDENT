Formate

Aufgrund unterschiedlicher Workflows kann es jedoch vorkommen, dass Export Files nicht eingelesen werden können. Wir bitten Sie daher, uns diese Files zu schicken, um sie zukünftig in der FOR-IDENT Plattform nutzbar zu machen. Vielen Dank! Exportformate von Bruker sind in Bearbeitung.

Folgende Formate werden von FOR-IDENT unterstützt (Details siehe Manual):
.cef (Agilent)
.mzML (Waters)
.txt (Sciex)
.xlsx / .csv (Thermo)

MasterView Software Benutzer können ein FOR-IDENT kompatible Datei erstellen, indem Sie folgende Datei txt-Export_MS2 auf Ihrem PC abspeichern und in der MasterView Software unter Einstellungen -> Tab Calculations als Report Vorlage auswählen.

Compound Discoverer Benutzer können folgende R Skripte, mit ausführlicher Anleitung verwenden, um ein FOR-IDENT kompatible .txt Datei zu erstellen.
