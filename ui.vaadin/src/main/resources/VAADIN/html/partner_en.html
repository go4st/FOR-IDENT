Formats

Due to diverse workflows it is possible that some export files cannot be read. We are kindly asking you to send us these files, to make them ready for importing in the future. Many thanks in advance! Export files from Bruker will be available soon.

The following formats are supported by the FOR-IDENT Platform (details within the manual):
.cef (Agilent)
.mzML (Waters)
.txt (Sciex)
.xlsx / .csv (Thermo)

MasterView Software user can generate a FOR-IDENT compatible file by saving the file txt-Export_MS2 on your PC and specify it in the MasterView Software within Settings in the tab calculations as the used the template.

Compound Discoverer Software user can use following R scripts containing a detailed tutorial to create the correct .txt export file.
